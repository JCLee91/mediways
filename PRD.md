# Mediways PRD (Product Requirements Document)
**메디웨이즈 - AI 의료광고 콘텐츠 생성 플랫폼**

---

## 📌 1. 프로젝트 개요

### 1.1 프로젝트명
- **Mediways 고도화 및 SEO 최적화 시스템 개발**
- **버전**: 2.0
- **최종 업데이트**: 2024년 12월

### 1.2 프로젝트 목표
의료기관이 법적 리스크 없이 효과적인 마케팅 콘텐츠를 제작할 수 있도록 AI 기반 플랫폼 고도화 및 SEO 최적화 기능 개발.

### 1.3 배경 및 필요성
- 대한민국 의료법 제56조 및 시행령 제23조 준수 의무화
- 의료광고 사전심의 제도 강화로 인한 콘텐츠 제작 어려움
- AI 기술 발전으로 자동화된 콘텐츠 생성 및 SEO 최적화 가능성 증대

### 1.4 타겟 사용자
- **1차 타겟**: 병원/의원 마케팅 담당자
- **2차 타겟**: 의료광고 대행사
- **3차 타겟**: 개인 의료진 및 병원장

## 🎯 2. 핵심 기능 요구사항

### 2.1 의료법 준수 자동 검수 시스템 ✅
**최우선 기능 - 100% 준수 보장**

- [x] **실시간 검수**: 콘텐츠 생성 시점 자동 법규 검사
- [x] **위험 표현 탐지**: 100+ 금지 표현 자동 필터링
- [x] **위험도 평가**: 고위험/중위험/저위험 등급 자동 분류
- [x] **자동 수정 제안**: 위반 표현 대체 문구 자동 추천
- [x] **검수 로그 관리**: 모든 검수 이력 저장 및 추적

### 2.2 SEO 최적화 시스템 ✅
**신규 개발된 핵심 기능**

#### 2.2.1 SEO 점수 산출 알고리즘
- **점수 범위**: 0-100점 (등급: 불량/개선필요/양호/우수/최고)
- **가중치 배분**:
  - 콘텐츠 품질: 40%
  - 키워드 최적화: 25%
  - 의료법 준수: 20%
  - 가독성: 15%

#### 2.2.2 분석 요소
- [x] **키워드 밀도**: 타겟 키워드 1-3% 최적화
- [x] **제목 최적화**: 30-60자 권장 길이
- [x] **메타 설명**: 120-160자 최적화
- [x] **구조화 데이터**: 의료 스키마 마크업
- [x] **가독성**: 문장 길이, 단락 구조 분석

### 2.3 AI 콘텐츠 생성 시스템 ✅

#### 2.3.1 지원 콘텐츠 유형
- [x] **블로그**: 리뷰형/정보형 2가지 템플릿
- [x] **SNS 게시물**: 인스타그램/틱톡/X/쓰레드
- [x] **YouTube 스크립트**: 전문의 톤앤매너
- [x] **카피라이팅**: 광고 카피 5개 자동 생성

#### 2.3.2 진료과목 특화 (30개+)
- 피부과, 치과, 성형외과, 소아과
- 내과, 외과, 산부인과, 안과
- 이비인후과, 정형외과, 신경외과
- 재활의학과, 정신건강의학과 등

### 2.4 블로그 SEO 분석 시스템 ✅
**완전 신규 개발 기능**

- [x] **RSS 피드 연동**: 실시간 블로그 모니터링
- [x] **자동 SEO 분석**: 개별 글 및 전체 평균 점수
- [x] **개선 제안**: 구체적 실행 방안 제공
- [x] **성과 추적**: 주간/월간 리포트 자동 생성

### 2.5 통합 대시보드 ✅

- [x] **실시간 모니터링**: 의료법 준수도, SEO 평균 점수
- [x] **분석 현황**: 처리된 콘텐츠 개수 및 상태
- [x] **최근 활동**: 분석 결과 및 개선 사항
- [x] **데이터 수집**: RSS 피드 연동 현황

## 🛠️ 3. 기술 스펙

### 3.1 Frontend
- **Framework**: Next.js 15.3.2 (App Router)
- **Language**: TypeScript 5.8.3
- **Styling**: Tailwind CSS 3.4.17
- **UI Components**: Radix UI 기반 커스텀 컴포넌트

### 3.2 Backend
- **Runtime**: Edge Runtime (Vercel)
- **Database**: Supabase PostgreSQL
- **Authentication**: Supabase Auth
- **AI Service**: OpenAI GPT-4o-mini API
- **Rate Limiting**: 커스텀 구현

### 3.3 주요 API 엔드포인트
- **POST** `/api/generate` - AI 콘텐츠 생성
- **POST** `/api/seo-analysis` - SEO 분석
- **GET** `/api/blog-analysis` - 블로그 분석
- **GET** `/api/dashboard` - 대시보드 데이터
